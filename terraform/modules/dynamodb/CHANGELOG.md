# Changelog

All notable changes to this module will be documented in this file.

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of DynamoDB module
- KMS encryption support (required)
- Point-in-time recovery
- TTL configuration
- Global and Local Secondary Index support
- DynamoDB Streams support
- Auto scaling for provisioned capacity mode
- Deletion protection option
- Common tags integration
